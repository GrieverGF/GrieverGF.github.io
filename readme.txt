Chuck Norris Jokes Web App

Este proyecto es una aplicación web simple que muestra chistes de Chuck Norris al hacer clic en un botón. Utiliza la API pública de Chuck Norris para obtener los chistes aleatorios.

Contenido del Proyecto:
- `index.html`: Archivo HTML principal que contiene la estructura de la página.
- `script.js`: Archivo JavaScript que realiza la lógica para obtener y mostrar los chistes de Chuck Norris.
- `README.txt`: Este archivo que proporciona información sobre el proyecto.
